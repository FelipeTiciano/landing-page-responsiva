Exemplo de histórico de commits (simulado) — execute localmente para reproduzir:

git init
git checkout -b develop
git add .
git commit -m "feat: initial accessible layout"
git checkout -b feature/gitflow
git commit -am "docs: add gitflow and README improvements"
git checkout develop
git merge --no-ff feature/gitflow -m "chore: merge feature/gitflow into develop"
git checkout -b release/1.0.0
git commit -m "chore(release): prepare release 1.0.0"
git checkout main
git merge --no-ff release/1.0.0 -m "chore(release): merge release 1.0.0 into main"
git tag -a v1.0.0 -m "Release v1.0.0"
